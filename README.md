# Turin 2026 Crash Course

Course materials for the 2026 Turin University Crash Course "ICT for Civic Data", part of the Master programme "ICT for Development and Social Good".

**Website:** https://clombion.github.io/Turin2026-crash-course/

## Contents

- `site/` — Course website (landing page, day pages, section pages, exam, resources)
- `slides/` — Reveal.js slide decks for each day (morning and afternoon sessions)
- `handouts/` — Self-paced review handouts (Foundations, Sections A–G)
- `demos/` — Interactive map demos built during the course (Senegal health facilities, Bordeaux, flood dashboard)
- `resources/` — Reference documents (RFPs, data pipeline overview, guides)
- `data/` — Source datasets (FloodArchive, Senegal floods)
- `scripts/` — Python scripts for data processing
- `pdfs/` — PDF exports of all presentations and the exam RFP
